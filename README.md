**GeoRed – MVP**

GeoRed es un MVP funcional que permite publicar y buscar servicios locales en un mapa interactivo. Su objetivo principal es validar si las personas publicarán servicios y si los usuarios realmente los utilizarán para encontrar opciones cercanas.

Nota: La aplicación puede tardar algunos segundos en cargar al abrir el sitio web, debido a las limitaciones del hosting utilizado para esta versión de prueba.

**Características principales**

- Publicación inmediata de servicios con ubicación geográfica.

- Visualización mediante pines en un mapa interactivo.

- Filtros por categoría y por radio de distancia.

- Contacto directo con el proveedor.

- Interfaz minimalista y de carga rápida.
